# Olympus 영어 학습 앱 — 사용 안내

EBS 2026 올림포스 영어독해(고1) 기출문제집에서 추출한 단어와 문장으로 학습할 수 있는 PWA(Progressive Web App)입니다.

## 실행 방법

### PC (가장 간단)
1. `olympus_eng.html` 파일을 컴퓨터에 다운로드
2. 더블클릭하면 브라우저에서 자동으로 열림
3. 끝.

### 휴대폰에 "앱처럼" 설치하기 (PWA)

이 앱은 PWA로 만들어져 있어서 휴대폰에 **진짜 앱처럼 설치**할 수 있습니다.

**iOS (iPhone/iPad)**
1. 파일을 iCloud Drive 등에 올린 후 Safari로 열기
2. 하단 공유 버튼 탭
3. "홈 화면에 추가" 선택
4. 홈 화면에 Olympus 앱 아이콘이 생성됨
5. 탭하면 전체 화면 앱처럼 실행 (브라우저 UI 없음)

**Android**
1. 파일을 Google Drive 등에 올린 후 Chrome으로 열기
2. 우측 상단 메뉴(⋮) 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. 홈 화면에 앱 아이콘 생성

> **주의**: PWA로 설치하려면 파일이 `https://`로 호스팅되어야 가장 안정적입니다. `file://`로 열어도 작동은 하지만 브라우저에 따라 "설치" 옵션이 안 보일 수 있습니다. 이런 경우엔 GitHub Pages, Netlify 등 무료 호스팅에 올려서 사용하면 완벽한 PWA가 됩니다.

## 폰트

이 앱은 다음 우선순위로 폰트를 사용합니다:

**한국어**
1. **한컴 말랑말랑체** (사용자 PC에 설치되어 있을 경우 자동 사용)
2. Gaegu (Google Fonts, 손글씨풍 둥근 폰트)
3. 맑은 고딕 (Windows 기본)
4. Apple SD Gothic Neo (Mac/iOS 기본)

**영어 디스플레이**: Jua (배민 주아체, 둥근 느낌) → 시스템 sans-serif

### 한컴 말랑말랑체를 사용하고 싶다면
[한컴 폰트 페이지](https://font.hancom.com)에서 한컴 말랑말랑체를 다운받아 PC에 설치하면 자동으로 적용됩니다. 라이선스상 웹폰트로 배포할 수 없어서 사용자 본인 PC에 설치하는 방식만 지원합니다.

### 난이도 기준
- **하** (578개): 일상 기본 어휘 — 예: good, way, need
- **중** (829개): 표준 학습 어휘 — 예: threat, creative, motivate
- **상** (239개): 고급/학술 어휘 — 예: organism, sophisticated

## 주요 기능

### 챕터별 학습
1. 단어 학습 (플래시카드) — 카드 뒤집기, 영↔한 모드 전환, 발음 듣기 (♪)
2. 단어 퀴즈 (영→한) — 4지선다, 챕터당 최대 20문제
3. 단어 퀴즈 (한→영) — 한국어 뜻 보고 영어 단어 고르기
4. 문장 학습 — 문법 포인트별 필터, 한국어 해석 토글
5. 문법 퀴즈 — 문장의 핵심 문법 포인트 맞추기

### 통계 & 약점 분석
- 학습 진행률 (단어·문장·챕터)
- 난이도별 학습 현황
- 약점 단어 자동 추적 → 자주 틀린 단어만 모아서 다시 학습 가능
- 연속 학습일 (streak)

### 키보드 단축키 (PC)
- Space — 플래시카드 뒤집기
- ← — "모르겠어요"
- → — "알아요"
- Esc — 뒤로 가기

## 데이터 저장

모든 학습 진행 상황은 브라우저 localStorage에 저장됩니다.
- 같은 브라우저에서 다시 열면 이어서 학습 가능
- 브라우저 데이터 삭제 시 진행상황도 사라짐
- 다른 기기/브라우저와 동기화는 되지 않음
- 통계 화면 하단의 "초기화" 버튼으로 모든 데이터 삭제 가능
